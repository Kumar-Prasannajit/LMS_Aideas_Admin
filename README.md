# 📚 LMS Admin Dashboard (Frontend)

A React-based admin dashboard for managing courses, modules, users, and enrollments.

---

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

---

### 2️⃣ Install dependencies

```bash
npm install
```

---

### 3️⃣ Run the development server

```bash
npm run dev
```

---

### 4️⃣ Open in browser

Visit:

```
http://localhost:5173
```

---

## 🛠 Tech Stack

* React (Vite)
* React Router DOM
* CSS

---

## 📂 Project Structure (Quick Overview)

```
src/
 ├── pages/        → Main screens (Courses, CourseDetails, etc.)
 ├── components/   → Reusable UI components (Sidebar, CourseCard)
 ├── layout/       → Layout wrapper (AdminLayout)
 ├── data/         → Static mock data (courses, modules)
```

---

## ✨ Features

* Course listing dashboard
* Course details with modules
* Enrolled users view
* Sidebar navigation
* Tab-based UI

---

## ⚠️ Note

This is a frontend-only project using static data.
No backend or API integration yet.
